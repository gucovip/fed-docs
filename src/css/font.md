# font

## 常用属性

```css
.font {
    /* 文字尺寸：14像素 */ 
    font-size: 14px;
    /* 文字颜色：红色 */ 
    color: red;
    /* 字体的粗细：加粗 */
    font-weight: bold;
    /* 文字对其方式：居中 */ 
    text-align: center;
    /* 文字下划线：清除 */ 
    text-decoration: none;
    /* 文字字符间距：2像素 */ 
    letter-spacing: 2px;
    /* 文字方向：从右向左 */ 
    direction: rtl;
}
```
